Selected work

★ Flagship

Najma (MyHoroscopeAI)

getnajma.com ↗

“Astrology that answers the question you actually asked.” Its core, Nujoom, is an agentic AI decision engine: ask it “best day to ask for a raise?” and it reads your birth chart, scores the timing out of 100, and explains why.

Grounded, not guessed — a deterministic scoring engine computes the numbers on Swiss Ephemeris chart math, and the LLM only narrates them (server-side validation stops it inventing anything). iOS in App Store review; web app live now.

  • Solo build
  • Agentic AI
  • React Native
  • Node.js
  • Supabase
  • Groq LLMs
  • Swiss Ephemeris

hiresort

hiresort.app ↗

“Rank applicants before you read every résumé by hand.” An AI screening tool that scores and ranks candidates against criteria you define — and shows its reasoning, each candidate's strengths and gaps, instead of an opaque match %.

Under the hood: local embeddings pre-filter candidates so only the best consume expensive LLM calls, and scoring runs as chunked, resumable jobs that survive serverless timeouts. Live at hiresort.app.

  • Next.js 14
  • TypeScript
  • Prisma / Postgres
  • Pluggable LLMs
  • Local embeddings
  • Stripe

Ahlein (WhatsApp receptionist)

Built · launch pending

A multi-tenant WhatsApp bot that books, reschedules, and manages appointments for salons & barbershops entirely through chat — in the customer's own language and dialect, with the owner kept in the loop.

Genuinely agentic: the LLM drives real tools (check availability, book, reschedule, hand off), transcribes voice notes natively across Gulf/Levantine Arabic, French & English, and owns a live calendar with two-way Google sync. Proven end-to-end on real WhatsApp.

  • Agentic AI
  • Python
  • FastAPI
  • Postgres / pgvector
  • Redis
  • Gemini
  • WhatsApp Cloud API
